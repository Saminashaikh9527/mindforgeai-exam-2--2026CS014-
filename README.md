# Student Performance Analytics

## Objective
This project is created to study student performance using Python and Pandas. It calculates readiness scores and analyzes student data.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## How to Run
1. Open the Jupyter Notebook.
2. Run each code cell one by one.
3. Check the outputs and results.

## Result Summary
- Loaded and displayed the student dataset.
- Calculated the readiness score for every student.
- Grouped students into Support, Developing, Strong, and Excellent.
- Calculated the average score for each department.
- Identified the top-performing student.
- Listed the students who need academic support.
